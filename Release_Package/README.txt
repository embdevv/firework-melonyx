GDPHYSX - Particle Physics Engine (Phase 1)
============================================

Engine Name: Melonyx

Group Members:
- Erica Mauriz Barundia

Description:
Melonyx is a particle-based physics engine designed to simulate fountain fireworks using 
OpenGL for rendering. It demonstrates core physics concepts including particle lifetime 
management, gravity simulation, and velocity/acceleration integration.

Features:
- Particle system with configurable spawn rates (1-1000+ sparks per second)
- Gravity simulation (10 m/s²)
- Particle lifespan management
- Dual camera projection modes (Orthographic and Perspective)
- Camera orbital rotation (WASD controls)
- Play/Pause simulation (Space key)
- Random particle colors and initial velocities
- Sphere-based particle rendering

Controls:
- 1 Key: Switch to Orthographic projection
- 2 Key: Switch to Perspective projection
- W/S Keys: Rotate camera vertically
- A/D Keys: Rotate camera horizontally
- Space: Play/Pause simulation

System Requirements:
- Windows 10/11 with OpenGL 3.3+ support
- Visual Studio 2019+ (for building from source)
